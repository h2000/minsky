Components in Minsky

Getting Started ›› Working with Minsky ››

There are three types of components in Minsky

  1. Mathematical operators such as plus (+), minus (-)
  2. Constants (or parameters) which are given a value by the user
  3. Variables whose values are calculated by the program during a simulation and depend on the values of constants and other variables; and
  4. Godley Tables, which define both financial accounts and the flows between them.
