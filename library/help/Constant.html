Constant

Getting Started ›› Components of the Program ›› Menu ›› Operations ››

Choosing this menu item brings up the Constant definition window:

A variable must be given a name--say Lab_Prod--and a value.

When you have provided a name and a value for the Constant, the so-named constant block will appear beneath the cursor. Move to where you want to locate it on the Canvas and click the left mouse button to place it.

A variable can also be given a range of possible values from Min to Max, with steps between these two extremes determined by the "Slider Step Size", which can be either an absolute amount or a percentage of the gap between Min and Max. Click here to see how to use this feature.
