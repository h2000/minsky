Variable

Getting Started ›› Components of the Program ›› Menu ›› Operations ››

Choosing this menu item brings up the variable definition window:

A variable must be given a name--say GDP in this example

The other two attributes--Value and Rotation--can be left blank. Normally no value will be provided for a variable, because its value will be determined by other entities in a model--for example, the variable GDP can be modeled as the variable Capital divided by the capital to output ratio.

After you provide a name for the variable and either press Enter or click on OK, the variable icon will appear beneath the cursor and can be placed anywhere on the canvas by clicking the mouse button.
